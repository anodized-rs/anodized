# Anodized Macros

You're probably looking for the [`anodized`](https://crates.io/crates/anodized) crate, which re-exports the proc macros provided by this crate, to work around the limitation that [a `proc-macro` crate can only export proc macros](https://doc.rust-lang.org/reference/linkage.html#r-link.proc-macro).
